The broadcast incremental power (BIP) method for minimum energy broadcasting algorithm is implemented in C++.

Enter the locations of the nodes in the "locations.txt" file as (x,y) pairs. Each location must be on one line. The first line must be the location of the source.

The program has been tested successfully on linux platforms.
